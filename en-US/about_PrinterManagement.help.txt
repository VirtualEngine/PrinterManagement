TOPIC
    about_PrinterManagement

SHORT DESCRIPTION
    The PrinterManagement module is a collection of Desired Configuration State (DSC) resources which are used for
    managing printers in a lab environment.

LONG DESCRIPTION
    The PrinterManagement resources can be used to create and share printers via owerShell Desired State
    Configuration (DSC). This module contains the following DSC resources:

    * PrinterDriver
    * Printer

KEYWORDS
    DSC, DSC Resource, Printer, Driver
